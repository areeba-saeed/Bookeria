## Run Locally

Clone the project

```bash
  git clone https://github.com/areeba-saeed/Bookeria.git
```

Go to the project directory

```bash
  cd Bookeria
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
